# Dollarama Inc. — Equity Valuation Model

**DOL.TO · Toronto Stock Exchange**

## Model Components

| # | Component | Description |
|---|---|---|
| 01 | **Historical Financials** | 11 years of P&L, Balance Sheet, and Cash Flow Statement |
| 02 | **WACC** | OLS beta regression (5Y weekly), Blume adjustment, CAPM, Damodaran ERP |
| 03 | **ROIC** | Value creation framework vs cost of capital |
| 04 | **3-Segment Revenue Forecast** | Bottom-up build: Canada + Australia (TRS) + Dollarcity (LatAm) |
| 05 | **DCF + FCFF** | Integrated model with mid-year convention |
| 06 | **Sensitivity Analysis** | 5×5 WACC vs terminal growth rate matrix |
| 07 | **Trading Comps** | 5 peer companies, EV/EBITDA and P/E multiples |
| 08 | **Football Field** | Valuation range across Bear / Base / Bull scenarios |
